🔑 Get Your Free YouTube API Key

1
Open Google Cloud Console:
Go to console.cloud.google.com — sign in with your Google account
2
Create a New Project:
Click "Select Project" at the top → "New Project" → Name it "RankIQ" → Click Create
3
Enable YouTube Data API v3:
Left menu → "APIs & Services" → "Library" → Search "YouTube Data API v3" → Click Enable
4
Create API Key:
Left menu → "APIs & Services" → "Credentials" → "Create Credentials" → "API Key"
5
Copy & Paste:
Copy the key that starts with "AIza..." and paste it in the RankIQ setup screen
