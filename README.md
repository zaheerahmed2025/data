# 📡 TubeIntel v2.0 — YouTube Channel Intelligence

Real YouTube Data API — seedha YouTube se data!

## 🔒 Security Note
API Key sirf tumhare browser localStorage mein save hoti hai — GitHub code mein kabhi nahi jaati. Safe hai!

## 🚀 GitHub Pages Deploy

1. github.com → New Repository → tubeintel → Public → Create
2. Upload files: index.html + css/style.css + js/app.js
3. Settings → Pages → Source: main → Save
4. Live: https://[username].github.io/tubeintel/

## 🔑 YouTube API Key (Free)

1. console.cloud.google.com → New Project
2. APIs & Services → YouTube Data API v3 → Enable
3. Credentials → API Key → Copy (AIza...)
4. TubeIntel mein paste karo

Free quota: 10,000 units/day = ~200 channel analyses daily
